# Libbox for iOS

A [Sing-Box](https://github.com/SagerNet/sing-box) build workflow for iOS. This repository produces an .xcframework called Libbox for integration into iOS apps.

## Overview

### - What is Sing-Box?

Sing-Box is a versatile proxy platform by SagerNet. This project focuses on building Sing-Box’s iOS component as a self-contained .xcframework.

### - Why Libbox?

Libbox is the name given to the compiled framework for Apple platforms. It helps developers integrate Sing-Box functionality in their iOS apps without manually setting up a complex build environment.
